Shopify
=======

`Shopify App`_ App for Shopify E-commerce software.

Get started quickly (after ``riptide setup``):
Run ``riptide cmd setup``, then check ``shopify-app.md`` for further steps.

.. _`Shopify App`: https://shopify.dev/apps

    :depth: 3
